# Playground (Vite + React)

Petit projet de test pour valider manuellement `@cappasoft/openrouter-model-selector` avant publication.

## Setup

Depuis la racine du repo :

```bash
cd examples/playground-vite-react
npm install
```

Optionnel (évite de coller la clé à chaque fois) :

```bash
export VITE_OPENROUTER_API_KEY="sk-or-v1-..."
```

## Run

```bash
npm run dev
```
